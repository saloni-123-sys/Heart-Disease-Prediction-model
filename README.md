# Heart Disease Prediction using Machine Learning

This project is a machine learning model to predict the presence of heart disease using patient data. The project is implemented in Python using Google Colab.

## Features

- Data collection and preprocessing using Pandas and NumPy.
- Logistic Regression model from scikit-learn for prediction.
- Model evaluation using accuracy score.
- Analysis and visualization of predictions.

## Libraries Used

- pandas
- numpy
- scikit-learn
